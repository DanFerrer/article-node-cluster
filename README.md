# Article Node Cluster

Proof of concept API that gets, downloads, and uploads files

  - Node
  - Express
  - Express-Limiter
  - Redis
  - Cluster
  - Mutler

## Getting Started

All dependecies are managed by npm
```
npm install
```


## Usage

run `npm start` to start the APi accross 12 cores on a single server on port 8080
